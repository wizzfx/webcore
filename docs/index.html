Git - Stuff

Some commands

  • - files can be: modified, stagged, commited
  • - mkdir ( create folder with chosen name )
  • - mv *.js ( move all .JS files to chosen folder )
  • - git config
  • - git config --list
  • - git --version
  • - git init ( create new empty repo within folder )
  • - git clone ( clone repo to folder )
  • - git pull ( download repo files )
  • - git push ( sending repo files )
  • - git add ( if you add files they are staged )
  • - git commit -m "description"
  • - git diff ( it can show changes in files )
  • - git diff ( shows changes for staged files )
  • - git rm ( remove files and will be showsn in staged files )
  • - ls ( show files all )
  • - rm ( remove file but will be not shown in staged files )
  • - mv ( move or actually RENAME file )
  • - git reset HEAD ( remove files from stagged state )
  • - git checkout -- file name ( we can recover delated files )
  • - git checkout -b branch name ( create branch and auto.. switch to it )
  • - git revert ( following with commit SHA code )
  • - git branch ( create branch following with branch name )

pushing new repo to empty repo on GitHub:

git remote add origin http://repourl
git push -u origin master
